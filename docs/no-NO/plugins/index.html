Installere plugins

NFive Package Manager (Pakkesystem)

NFive Package Manager, eller mer kjent som "nfpm" brukes til å installere plugins direkte inn i NFive. Du kan bruke nfpm search eller besøke plugin hub for å finne plugins du kan være interessert i!

Eksempel

nfpm install <plugin>
nfpm install <plugin>@<version>

Første gang du installerer plugins?

NFive har også gitt ut flere plugins, om dette er første gang du benytter NFive anbefaler vi deg å prøve og installere de nedenfor

Start (Starte)

nfpm install NFive/plugin-start

Loading Screen (Lasteskjerm)

nfpm install NFive/plugin-loadingscreen

Session Manager (Økt manager)

nfpm install NFive/plugin-sessionmanager
