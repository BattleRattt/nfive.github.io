Instalando Plugins

NFive Package Manager (Gerenciador de Pacotes)

NFive Package Manager, mais conhecido como "nfpm" é usado para instalar plugins diretamente no NFive. Você pode usar nfpm search ou visitar o plugin hubpara achar plugins no qual você pode se interessar!

Exemplo

nfpm install <plugin>
nfpm install <plugin>@<version>

Primeira vez instalando plugins?

NFive tem vários plugins lançados, se for sua primeira vez usando NFive recomendamos experimentar e instalar estes abaixo

Start (Inicializado)

nfpm install NFive/plugin-start

Loading Screen (Tela de Carregamento)

nfpm install NFive/plugin-loadingscreen

Session Manager (Gerenciador de sessão)

nfpm install NFive/plugin-sessionmanager
