Configurando Servidor

NFive tem como objetivo ser muito fácil de configurar, assim você pode rapidamente instalar e rodar um servidor.

Primeiros passos

A maneira mais fácil de instalar um servidor FiveM e configurar NFive é usar a ferramenta nfpm:

  1. Baixe a ultima versão do nfpm: build de desenvolvimento

    nfpm.exe não precisa ser instalado e pode ser executado em qualquer lugar.

  2. Coloque-o em uma pasta vazia, nela que será instalado o servidor.

  3. Execute o comando de instalação nfpm setup e responda o que for pedido.

    nfpm funciona da mesma forma tanto no Linux quando no macOS se você usar oMono: mono nfpm.exe setup

Verifique sua instalação

A pasta do seu servidor deve conter a estrutura de arquivos semelhante:

root-directory
├── FXServer.exe
|── server.cfg
└── resources
    └── nfive
        ├── __resource.lua
        ├── nfive.yml
        ├── nfive.lock
        └── config
            ├── nfive.yml
            └── database.yml

Conexão do banco de dados

NFive precisa de uma conexão de banco de dados, para configurar você precisa colocar as informações no arquivo config/database.yml. Linux pode exigir alguma etapas adicionais, você pode precisar executar ./mysql_secure_instal e usar uma senha diferente dependendo da sua distribuição. <---- de uma pesquisa no Google sobre como corrigir isso. 2. Em uma instalação da MariaDB padrão no windows não é definida uma senha, por isso será executado normalmente usando o exemplo do arquivo config/database.yml abaixo:

# MySQL database connection
connection:
  host: db
  port: 3306
  database: fivem
  user: root
  password:
  charset: utf8mb4
  logging: false

migrations:
automatic: true

Iniciando o servidor

Depois de executar o comando de instalação você terá um servidor de FiveM em branco com o NFive. Para iniciar o servidor:

  1. No pasta principal execute o comando nfpm start.

    Você verá uma janela do servidor FiveM aberta e capaz de se conectar ao servidor a partir do seu client FiveM usando a guia "Direct Connect" (Conexão direta) e inserir o seu IP.

Se o servidor for iniciado corretamente e você conseguiu conectar, você pode seguir para próximos passos na configuração e instalação dos plugins, caso contrario olhe em "solução de problemas".
