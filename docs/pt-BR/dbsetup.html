Configurando Database

NFive makes use of a MySQL Database. Esse guia usará a MariaDB que foi recomendada e testada.

Primeiros passos

Windows Visite MariaDB. Baixe e instale a ultima versão.

Linux: Instale pelo seu gerenciador de pacotes (yum/apt etc)

Docker: NFive vem com um arquivo docker-compose que contem e usa MariaDB.

Configurando

NFive precisa ser totalmente compatível com Unicode, para isso será necessário pré-configurar manualmente o "character set" padrão do seu servidor MySQL: 1. character-set-server: utf8mb4 2. collation-server: utf8mb4_unicode_520_ci

Windows

Adicione embaixo de [mysqld] dentro de my.ini exemplo:

[mysqld]
character-set-server=utf8mb4
collation-server=utf8mb4_unicode_520_ci

O aquivo está localizado em C:\Program Files\MariaDB *version*\data\my.ini

Linux

Adicione embaixo de [mysqld] Adicione embaixo de my.cnf exemplo:

[mysqld]
character-set-server = utf8mb4
collation-server = utf8mb4_unicode_520_ci

O aquivo está localizado em /etc/mysql/my.cnf

Docker

Como MariaDB já vem inclusa no docker-compose, não é necessário fazer a pré-configuração a menos que esteja sendo usado seu próprio arquivo docker-compose. Nesse caso, basta copiar a configuração fornecida.
