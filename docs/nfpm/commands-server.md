## Reference

### `setup`

Install a new FiveM server and configures it to run NFive.

**Example**

```bash
nfpm setup
```

---

### `start`

Starts the FiveM server process in a new window.

**Example**

```bash
nfpm start
```

---

### `self-update`

Downloads and updates the newest NFive Plugin Manager (nfpm)

**Example**

```bash
nfpm self-update
```
